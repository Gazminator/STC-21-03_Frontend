По правилам тег дожен быть закрыт символом "/", однако, неокторые тэги, вроде <BR> можно не закрывать слэшем, что более
продуктивно, поскольку сокращает, пусть и не значительно, время загрузки страницы.

Атрибут.

Атрибут - это специальные ключевые слова внутри тега, которые придают ему дополнительное значение и несут некоторую информацию.
Например, атрибуты src и alt внутри тега img, где атрибут scr указывает местораположение файла изображения (src="image.jpg"), а тег
alt указывает браузеру выводимый текст вместо изображения в случае,если изображение отсутствует в каталоге
<img src="image.jpg" alt="Абырвалг!" >:

Элементом разметки выступает конструкция, содержащая атрибут и его значение, обрамляющие контент:

<a href="url"> текст </a>:

Структура HTML текста

< !DOCTYPE html > -- Используется для указания браузеру на какой версии HTML'a размечена страница. Всегда начинается с "!" и указанием "html"
< HTML > -- Код пишется в рамках этого тега
< HEAD > -- Является не обязательным с точки зрения стандартов, но практически всегда используется в качестве корневого элемента
для набора системных мета-тегов: < meta > задаёт выводимую в поисковике информацию, < title > указывает заголовок страницы (отображается на вкладке браузера)
< link > < script > позволяют подключать внешине ресурсы.
< /HEAD >
< BODY > -- Тег, в предалах которого размещается контент страницы. < /BODY >
< /HTML >
